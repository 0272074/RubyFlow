# RubyFlow (Wordフリガナ自動付与ツール)

Wordファイル（.docx）に、元のレイアウトや書式を一切崩すことなく、**自動で正確なフリガナ（ルビ）を付与**するモダンなWeb＋Pythonアプリケーションです。

## ✨ 主な特徴
- **完全なレイアウト維持**: Wordの厳格なOOXMLスキーマに準拠してネイティブなルビタグ（<w:ruby>）を生成します。
- **元の文字装飾を保持**: 太字、斜体、フォント色などの文字装飾をルビ付与後もそのまま引き継ぎます。
- **高度な送り仮名処理**: pykakasi を用いた形態素解析により、「過ぎ」に対して「すぎ」と全体にルビを振るのではなく、「過（す）ぎ」のように厳密に漢字部分のみにルビを割り当てます。
- **表（テーブル）対応**: ドキュメントの本文だけでなく、表（テーブル）の中に記述されたテキストにも漏れなくフリガナを付与します。
- **モダンなUI**: Next.js、Tailwind CSS、Shadcn UIを用いた直感的なドラッグ＆ドロップ画面を採用しています。

## 🚀 インストール方法
`ash
git clone https://github.com/yourusername/rubyflow.git
cd rubyflow/rubyflow-web
npm install
pip install python-docx pykakasi
npm run dev
`

## 🎮 使い方
1. http://localhost:3000 を開きます。
2. 画面中央のエリアにフリガナを振りたい Wordファイル（.docx）をドラッグ＆ドロップします。
3. 「フリガナを付与して保存」ボタンをクリックします。
4. 次のファイルを処理したい場合は「ファイルクリア」ボタン（×）を押して新たなファイルを追加してください。

> **💡 Wordの「網掛け」表示について**
> 処理後のファイルをWordで開いた際、ルビの背景がグレーの「網掛け」になっている場合があります。これはWordの「フィールドの網掛け」という表示設定によるものです（印刷時には出力されません）。
> 画面上から非表示にするには：
> Wordの[ファイル] ＞ [オプション] ＞ [詳細設定] ＞ [構成内容の表示] ＞ [フィールドの網掛け] を「表示しない」 に設定してください。
